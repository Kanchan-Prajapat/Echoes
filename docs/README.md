# Echoes Documentation

Welcome to the Echoes documentation.

This directory contains the technical documentation,
architecture decisions, design system, API references,
database schema, and product philosophy.

## Structure

- PRODUCT_PHILOSOPHY.md
- ROADMAP.md
- DESIGN_SYSTEM.md
- DATABASE.md
- API.md
- CHANGELOG.md
- decisions/

Every major engineering decision is documented inside
the decisions folder as an Architecture Decision Record (ADR).

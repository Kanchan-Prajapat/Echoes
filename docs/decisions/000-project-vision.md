# ADR-001: Project Vision

Status: Accepted

Date: 2026-07-02

---

## Context

Why are we building Echoes?

Many people store memories across multiple platforms such as Instagram,
Google Photos, WhatsApp, and cloud storage.

These platforms focus on storage or sharing rather than helping users
relive meaningful moments.

We wanted to create a private memory vault centered around storytelling.

---

## Decision

Echoes will be developed as a personal memory application rather than
a social media platform.

The application will not include:

- Followers
- Likes
- Comments
- Public profiles
- Recommendation algorithms

The primary focus will remain on preserving and reliving memories.

---

## Consequences

Positive

• Clear product identity

• Better user privacy

• Simpler architecture

Negative

• Less emphasis on social interaction

• Reduced viral growth potential

These trade-offs are acceptable because they align with the product vision.
